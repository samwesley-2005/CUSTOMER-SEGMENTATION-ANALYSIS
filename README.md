Project Overview :-
This project focuses on analyzing customer data to identify distinct customer segments based on spending behavior and characteristics.
The objective was to uncover patterns that support targeted marketing strategies and data-driven business decisions.

Objectives
- Segment customers based on spend patterns
- Analyze customer distribution
- Compare average spending behavior
- Evaluate location-based segmentation trends

Analysis Performed
 Customer Segmentation  
 Spend Analysis  
 Distribution Analysis  
 Comparative Metrics  
 Behavioral Insights

Key Visualizations
- Segment Distribution
- Average Spend per Segment
- Segment Count by Location

Key Insights
 Identified high-value customer segments  
 Analyzed spending differences across segments  
 Detected location-based customer patterns  
 Generated segmentation-driven business interpretation
